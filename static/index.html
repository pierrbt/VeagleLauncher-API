Découvez Griff, le nouveau launcher par l'équipe VeagleDev

Découvrez le tout nouveau launcher créé par Veagle, il fait suite au premier launcher créé il y a quelques mois mais avec un code entièrement réimaginé et avec un système bien plus élaboré pour l'installation des jeux.

Page principale

Les points forts :

  1. Facile d'utilisation
  2. Interface jolie et responsive
  3. Bibliothèques privées
  4. Grande bibliothèque publique
  5. Application open-source
  6. Équipe de développement active
  7. Technologies récentes (voir dessous)

Les points faibles :

  1. Jeux payants non-dispo sur la version gratuite
  2. Utilisation d'Electron qui peut être un peu long
  3. Utilisation d'une archive unique .7z
  4. Application encore en développement
  5. Encore en version de test

Open-source

C'est quoi l'open-source ?

L'open-source désigne un logiciel dont le code source est disponible publiquement et auquel chacun y a accès pour l'adapter à son besoin, l'améliorer et cela créé donc une collaboration sans but lucratif entre programmeurs afin de créer un produit fini fonctionnel et auquel chacun peut ajouter des modifications et ajouts. Pour en apprendre plus, visitez la page Wikipedia

Notre projet

Ce projet est un projet open-source hébergé sur GitHub par @pierrbt. Il a été programmé par l'équipe VeagleDev ainsi que par les collaborateurs. Cette application étant encore en développement, toutes les propositions ainsi que les ajouts/changements de code sont bienvenus. Vous pouvez ainsi accéder à l'entièreté du code pour comprendre comment le Launcher fonctionne et pour en vérifier par exemple la sécurité.
